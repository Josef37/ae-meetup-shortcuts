# macOS

* Search the toolbar: (US) <kbd>‚åò</kbd>+<kbd>?</kbd>, (DE) <kbd>‚åò</kbd>+<kbd>/</kbd>
* Typical keyboard shortcuts reference: (US) <kbd>‚åò</kbd>+<kbd>/</kbd>

* Faster desktop switching  
  <kbd>‚åò</kbd>+<kbd>‚á•</kbd> or <kbd>‚åÉ</kbd>+<kbd>1/2/...</kbd>
